# Prono Club

Prono Club vous permet de pronostiquer les résultats de matchs de football.

Vous pouvez rejoindre une ligue publique, avec toute l'administration des matchs assurée par l'équipe Prono Club.

Vous pouvez aussi créer une ligue privée, et vous aurez alors la responsabilité d'inviter vos amis, d'administrer les matchs, de calculer les points...

## Connexion et enregistrement

Pour vous enregistrer vous avez besoin :
- d'un mail (qui sera votre identifiant de connexion)
- d'un pseudo
- d'un mot de passe

Une fois enregistré, vous vous connectez avec votre mail (votre identifiant) et votre mot de passe.

Vous arrivez alors sur la page d'accueil.

## Choisissez une ligue et Go !

Dans la barre de navigation, vous pouvez voir une liste de sélection et un bouton Go.
Quand vous êtes engagés dans un certain nombre de ligues, cette liste de sélection contient les ligues dans lesquelles vous êtes engagés.
Vous pouvez alors choisir la ligue pour laquelle vous voulez jouer, puis cliquer sur le bouton Go.
Vous serez alors redirigés vers la page des pronostics de cette ligue.

Si vous êtes engagés dans aucune ligue, vous pouvez :
- rejoindre une ligue publique
- créer une ligue privée 

Quand vous êtes sur la page des pronostics d'une ligue, vous pouvez :
- pronostiquer les matchs "ouverts"
- voir les matchs pour lesquels vous avez fait un pronostic et qui sont actuellement clôturés
- voir ce que les autres participants ont pronostiqué pour les matchs clôturés
- voir le classement de la ligue
- voir votre historique (comment vous avez gagné vos points)

## Mon Compte

### Mon mot de passe

Cette page vous permet de changer votre mot de passe.

### Ma ligue par défaut

Cette page vous permet de positionner votre ligue par défaut.
Quand vous vous connectez, si vous avez une ligue par défaut, vous êtes automatiquement redirigé vers la page des pronostics de cette ligue.

### Mes invitations

Cette page vous permet de consulter les invitations que vous avez reçues.
Vous pouvez alors accepter ou décliner ces invitations.

Si vous acceptez, vous rejoignez la ligue concernée par cette invitation.
Si vous déclinez, cette invitation est supprimée.

### Rejoindre une ligue publique

Cette page vous permet de rejoindre une ligue publique.
Les ligues publiques sont gérées par l'équipe Prono Club.
Tout le monde peut rejoindre et pronostiquer dans une ligue publique.

### Inviter des amis

Cette page vous permet d'inviter des amis à la ligue à laquelle vous êtes en train de jouer (celle qui est sélectionnée dans la barre de navigation).
Vous pouvez inviter jusqu'à 5 amis via leur adresse mail.

## Administration

### Créer une ligue publique

Cette page est réservée aux admins Prono Club.
Elle permet de créer une nouvelle ligue publique.

### Créer une ligue privée

Cette page permet de créer une nouvelle ligue privée.
Il faut rentrer un nom de ligue.
On peut aussi positionner une ligue publique dont la ligue privée héritera des matchs.
Une ligue privée qui hérite des matchs d'une ligue publique n'a pas besoin d'administration.
Toute l'administration des matchs sera faite par l'équipe Prono Club dans la ligue publique.
Les ligues privées peuvent uniquement être rejointes sur invitation.

### Administrer la ligue

Cette page permet de changer le nom de la ligue.
Elle est réservée aux admins Prono Club pour les ligues publiques, et aux admins de la ligue pour les ligues privées.

### Administrer les matchs

Cette page permet d'administrer les matchs de la ligue.
Elle est réservée aux admins Prono Club pour les ligues publiques, et aux admins de la ligue pour les ligues privées.

Voici le cycle de vie d'un match dans une ligue :

1- Créer le match
Une fois créé, le match a 2 équipes, pas de score, et est fermé.
Tant qu'il est fermé, personne ne peut pronostiquer ce match.

2- Ouvrir le match
Une fois ouvert, les participants peuvent pronostiquer ce match.

3- Clôturer le match
Une fois clôturé, personne ne peut pronostiquer ce match.
Le match est probablement en train de se jouer à Anfield Road, Geoffroy-Guichard, ou Santiago Bernabeu.

4- Editer le match, pour positionner le score
Quand vous connaissez le score exact du match, vous pouvez le saisir en éditant le match.

5- Calculer les points
Vous pouvez alors calculer les points pour chaque joueur ayant pronostiqué ce match :
- 3 points pour un score exact (vous avez pronostiqué que le Real Madrid gagnerait 2-0 contre Getafe et c'est ce qui s'est exactement passé)
- 1 point pour un score correct (vous avez pronostiqué que Monaco gagnerait 1-0 contre Lyon, mais le score exact a été 3-1 pour Monaco)
- 0 point for un résultat complètement faux (vous aviez pronostiqué que la Sampdoria gagnerait contre l'Inter 1-0 mais il ya eu un nul 1-1)

Une fois que les points ont été calculés, le classement de la ligue a été mis à jour. 

### Administrer les participants

Cette page permet d'administrer les participants de la ligue.
Elle est réservée aux admins Prono Club pour les ligues publiques, et aux admins de la ligue pour les ligues privées.

Sur cette page on peut recalculer le classement completd e cette ligue.
On peut également notifier les participants (envoi d'un mail avec le sujet et le texte saisi).
